About Leash

What is this?

Leash is a site developed using Flask as part of my work toward a Nanodegree in Full Stack Web Development offered by Udacity. The site enables a network of animal shelters to display pets (specifically puppies in this project) to potential adopters. The site offers pretty comprehensive functionality, including:

  • Ability to add, edit and remove puppies and shelters from the database.
  • Ability to self-register as an adopter.
  • Ability to adopt a puppy online.
  • Function to evenly distribute puppies across shelters.
  • Administrative functions to add, edit and remove users.
  • Three distinct user roles:
    • Adopter/general public
    • Shelter operator
    • Administrator
  • Ability to upload photos of puppies.
  • Email functions that enable registering as a user and resetting a forgotten password.
  • Integration with Gravatar.
  • Complete set of unit tests to ensure a working application.
  • Custom error logging.

This is a working app and is open source, so feel free to use!
